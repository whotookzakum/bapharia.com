# Rewards-related files

## Server
- Rewards for quests, adventure boards, etc: `/rewards.json`
- Lottery rewards: `/master_reward_lottery_groups.json`

### Variables
`reward_type`
- 0: luno
- 2: exp
- 3: item
- 4: weapon
- 5: costume
- 7: mount
- 9: gesture/emote
- 10: stamp
- 11: imagine recipe
- 14: achievement
- 15: liquid memory
- 19: token
- 20: crafting recipe
- 27: crafting recipe set
- 28: adventure board
- 30: warehouse ability
- 33: lottery (RNG)
